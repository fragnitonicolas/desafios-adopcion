# Logos

Colocar aquí los archivos de logos con exactamente estos nombres:

- `scba.png` — Escudo/logo de la SCBA (Suprema Corte de la Pcia. de Buenos Aires)
- `ajb.png`  — Isologo de la AJB (Asociación de Jueces de Buenos Aires)

Formatos aceptados: PNG con fondo transparente (preferido) o SVG renombrado a .png.
El CSS aplica `filter: brightness(0) invert(1)` sobre ambos logos en el header
para adaptarlos al fondo oscuro — si el logo ya es blanco sobre transparente, quitar ese filtro.
